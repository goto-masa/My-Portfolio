story

このサイトを作った目的

  • 自分の作ったポートフォリオサイトに興味を持っていただき、作品集やスキルセットを見ていただきたいという目的で作成しました。
  • その為にはどのようにすればいいか考えた結果、読みやすさ綺麗さを整えてシンプルなデザインに統一し、スッキリとしたデザイン設計にしました。
  • webサイトは商品や作品を他人に知ってもらう術として今や欠かせない手段の一つであり、これからも流行はどんどん変わっていくと思います。その一つの手段としてのプログラミングを価値として提供していけたらいいなと思っています。

Skill

  • HTML
    HTMLを使ってサイトの基本的な骨組みを設計します。読みやすくて理解しやすく修正のしやすいコード作りを心掛けます。
  • CSS
    CSSを使ってwebページの装飾を行います。色や形、配置を決める言語です。レスポンシブにも対応できるサイトを制作いたします。
  • JavaScript
    動きのある表現のサイト作りも対応します。簡単な動きを実装することができますのでお任せください。
  • jQuery
    JavaScriptをさらに使いやすくするために設計されたライブラリを使ってよりスムーズにサイト制作を行います。
  • Photoshop
    写真画像のレタッチや切り抜きやバナー作成、webサイトのデザインを制作します。PNG形式やJPEG形式、PDF形式での書き出しも可能です。
  • Illustrater
    Illustraterを使って簡単なロゴ作成や文字のデザイン作成も行います。
