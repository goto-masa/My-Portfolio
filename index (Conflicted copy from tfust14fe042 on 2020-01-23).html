story

このサイトを作った目的

  • 自分の作ったポートフォリオサイトに興味を持ってもらいたい、作品集やスキルセットを見ていただきたいという目的で作成しました。
  • その為にはどのようにすればいいか考えた結果、読みやすさ綺麗さはもちろん、読み進めたくなるデザインでなければならないと考えサイト全体の色を少ない色で抑え先進的な印象を与えるデザインで設計しました。
  • このサイトを見た方が、このサイトを作っている人にHP制作を頼んで見ようかな、、そう思ってもらえるようなサイトデザインを心掛けました。ターゲットは若い20〜35歳くらいの方で自分のサイトを持ってみたい方を想定ました。

Skill

  • HTML
    HTMLを使ってサイトの基本的な骨組みを設計します。読みやすくて理解しやすく修正のしやすいコード作りを心掛けます。
  • CSS
    CSSを使ってwebページの装飾を行います。色や形、配置を決める言語です。レスポンシブにも対応できるサイトを制作いたします。
  • JavaScript
    動きのある表現のサイト作りも対応します。簡単な動きを実装することができますのでお任せください。
  • jQuery
    JavaScriptをさらに使いやすくするために設計されたライブラリを使ってよりスムーズにサイト制作を行います。
  • Photoshop
    写真画像のレタッチや切り抜きやバナー作成、webサイトのデザインを制作します。PNG形式やJPEG形式、PDF形式での書き出しも可能です。
  • Illustrater
    Illustraterを使って簡単なロゴ作成や文字のデザイン作成も行います。
